HTML 레이아웃 (Layout)


레이아웃(layout) 이란 특정 공간에 여러 구성 요소를 보기 좋게 효과적으로 배치하는 작업
웹 페이지의 레이아웃은 웹 사이트의 외관을 결정짓는 중요한 요소로, 직관적인 UI/UX와 관련해서도 매우 중요하다
